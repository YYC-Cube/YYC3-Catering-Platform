## 1. 系统概述

### 1.1 系统简介

YYC³ 餐饮管理平台是一款专为餐饮行业打造的全方位管理系统，集成了商品管理、订单管理、库存管理、客户管理、营销管理、财务管理、数据报表等核心功能，帮助餐饮企业实现数字化转型，提升运营效率，降低运营成本。

### 1.2 核心功能

- **商品管理**：商品上架、下架、分类、库存管理
- **订单管理**：订单接收、确认、配送、退款
- **库存管理**：库存调整、预警、盘点
- **客户管理**：客户信息、标签、等级管理
- **营销管理**：优惠券、促销活动、会员管理
- **财务管理**：支付、提现、账单管理
- **数据报表**：销售报表、库存报表、客户报表
- **系统设置**：店铺设置、权限管理、通知设置

### 1.3 技术特点

- **响应式设计**：支持PC、平板、手机多端访问
- **实时同步**：数据实时同步，确保信息准确
- **智能推荐**：基于AI的智能推荐系统
- **安全可靠**：采用银行级安全加密技术
- **易于使用**：简洁直观的操作界面

## 3. 账号管理

### 3.1 个人信息

#### 3.1.1 查看个人信息

1. 点击页面右上角的用户头像
2. 在下拉菜单中点击"个人中心"
3. 进入"个人信息"页面
4. 显示个人信息

#### 3.1.2 修改个人信息

1. 在"个人信息"页面点击"编辑"按钮
2. 修改需要更新的信息：
   - 昵称
   - 邮箱
   - 手机号
   - 性别
   - 生日
   - 头像
3. 点击"保存"按钮
4. 系统提示保存成功

### 3.2 账号安全

#### 3.2.1 修改密码

1. 进入"个人中心"页面
2. 点击"账号安全"标签页
3. 点击"修改密码"按钮
4. 输入原密码
5. 输入新密码
6. 确认新密码
7. 点击"确定"按钮
8. 系统提示修改成功

**密码要求**：

- 长度8-20位
- 包含字母和数字
- 不能与原密码相同
- 不能包含用户名

#### 3.2.2 绑定手机号

1. 进入"个人中心"页面
2. 点击"账号安全"标签页
3. 点击"绑定手机号"按钮
4. 输入手机号（11位）
5. 点击"获取验证码"按钮
6. 接收短信验证码（6位数字）
7. 输入验证码
8. 点击"确定"按钮
9. 系统提示绑定成功

#### 3.2.3 绑定邮箱

1. 进入"个人中心"页面
2. 点击"账号安全"标签页
3. 点击"绑定邮箱"按钮
4. 输入邮箱地址
5. 点击"发送验证码"按钮
6. 查收邮件，获取验证码
7. 输入验证码
8. 点击"确定"按钮
9. 系统提示绑定成功

### 3.3 消息通知

#### 3.3.1 查看消息

1. 点击页面右上角的"消息"图标
2. 进入"消息中心"页面
3. 显示所有消息列表

#### 3.3.2 消息类型

- **订单消息**：订单相关通知
- **支付消息**：支付相关通知
- **促销消息**：促销活动通知
- **系统消息**：系统相关通知

#### 3.3.3 消息操作

- **标记已读**：点击消息标记为已读
- **删除消息**：点击删除按钮删除消息
- **全部已读**：点击"全部已读"按钮标记所有消息为已读

## 4. 商品管理

### 4.1 商品列表

#### 4.1.1 查看商品列表

1. 点击左侧导航菜单的"商品管理"
2. 进入"商品列表"页面
3. 显示所有商品

#### 4.1.2 搜索商品

在商品列表页面，可以通过以下方式搜索商品：

- **商品名称**：输入商品名称搜索
- **商品分类**：选择商品分类筛选
- **商品状态**：选择商品状态筛选（上架/下架）
- **创建时间**：选择时间范围筛选

#### 4.1.3 商品列表字段

| 字段     | 说明                   |
| -------- | ---------------------- |
| 商品ID   | 商品唯一标识           |
| 商品名称 | 商品名称               |
| 商品分类 | 商品所属分类           |
| 商品价格 | 商品售价               |
| 商品库存 | 当前库存数量           |
| 商品状态 | 上架/下架              |
| 创建时间 | 商品创建时间           |
| 操作     | 编辑、删除、上架、下架 |

### 4.2 添加商品

#### 4.2.1 添加商品步骤

1. 在商品列表页面点击"添加商品"按钮
2. 进入"添加商品"页面
3. 填写商品信息：
   - **基本信息**：商品名称、商品分类、商品价格、商品库存
   - **商品描述**：商品详细描述
   - **商品图片**：上传商品图片
   - **商品规格**：设置商品规格（可选）
4. 点击"保存"按钮
5. 系统提示添加成功

#### 4.2.2 商品信息填写规范

**基本信息**：

- 商品名称：2-200个字符，不能包含特殊字符
- 商品分类：必填，从下拉列表中选择
- 商品价格：必填，必须大于0，保留两位小数
- 商品库存：必填，必须大于等于0，整数

**商品描述**：

- 0-5000个字符
- 支持富文本编辑
- 可以插入图片、视频

**商品图片**：

- 支持 JPG、PNG、GIF 格式
- 建议尺寸 800x800 像素
- 单张图片不超过 5MB
- 最多上传 9 张图片

**商品规格**：

- 规格名称：如颜色、尺寸
- 规格值：如红色、蓝色、M、L
- 规格价格：不同规格可以设置不同价格
- 规格库存：不同规格可以设置不同库存

### 4.3 编辑商品

#### 4.3.1 编辑商品步骤

1. 在商品列表中找到需要编辑的商品
2. 点击"编辑"按钮
3. 进入"编辑商品"页面
4. 修改商品信息
5. 点击"保存"按钮
6. 系统提示保存成功

#### 4.3.2 编辑限制

- 商品ID不能修改
- 商品创建时间不能修改
- 商品销量不能修改（系统自动统计）

### 4.4 上架/下架商品

#### 4.4.1 上架商品

1. 在商品列表中找到需要上架的商品
2. 点击"上架"按钮
3. 系统提示确认操作
4. 点击"确定"按钮
5. 系统提示上架成功
6. 商品状态变为"上架"

#### 4.4.2 下架商品

1. 在商品列表中找到需要下架的商品
2. 点击"下架"按钮
3. 系统提示确认操作
4. 点击"确定"按钮
5. 系统提示下架成功
6. 商品状态变为"下架"

#### 4.4.3 批量上架/下架

1. 在商品列表中勾选需要操作的商品
2. 点击"批量操作"按钮
3. 选择"批量上架"或"批量下架"
4. 系统提示确认操作
5. 点击"确定"按钮
6. 系统提示操作成功

### 4.5 删除商品

#### 4.5.1 删除商品步骤

1. 在商品列表中找到需要删除的商品
2. 点击"删除"按钮
3. 系统弹出确认对话框
4. 点击"确定"按钮
5. 系统提示删除成功

#### 4.5.2 删除限制

- 有未完成订单的商品不能删除
- 建议先下架商品，确认无订单后再删除

### 4.6 商品分类

#### 4.6.1 查看商品分类

1. 点击左侧导航菜单的"商品管理"
2. 点击"商品分类"子菜单
3. 进入"商品分类"页面
4. 显示所有商品分类

#### 4.6.2 添加商品分类

1. 在商品分类页面点击"添加分类"按钮
2. 填写分类信息：
   - 分类名称（必填）
   - 父分类（选填）
   - 排序序号（选填）
   - 分类图标（选填）
3. 点击"保存"按钮
4. 系统提示添加成功

#### 4.6.3 编辑商品分类

1. 在商品分类列表中找到需要编辑的分类
2. 点击"编辑"按钮
3. 修改分类信息
4. 点击"保存"按钮
5. 系统提示保存成功

#### 4.6.4 删除商品分类

1. 在商品分类列表中找到需要删除的分类
2. 点击"删除"按钮
3. 系统弹出确认对话框
4. 点击"确定"按钮
5. 系统提示删除成功

**删除限制**：

- 有子分类的分类不能删除
- 有商品的分类不能删除

## 5. 订单管理

### 5.1 订单列表

#### 5.1.1 查看订单列表

1. 点击左侧导航菜单的"订单管理"
2. 进入"订单列表"页面
3. 显示所有订单

#### 5.1.2 搜索订单

在订单列表页面，可以通过以下方式搜索订单：

- **订单号**：输入订单号搜索
- **用户昵称**：输入用户昵称搜索
- **用户手机号**：输入用户手机号搜索
- **订单状态**：选择订单状态筛选
- **支付状态**：选择支付状态筛选
- **创建时间**：选择时间范围筛选

#### 5.1.3 订单状态

| ------ | ------------------------ |

| 待配送 | 订单已支付，等待配送     |
| 配送中 | 订单正在配送中           |
| 已完成 | 订单已完成               |
| 已取消 | 订单已取消               |

| 字段       | 说明                                   |

| 用户昵称   | 下单用户昵称                           |
| 用户手机号 | 下单用户手机号                         |
| 商品信息   | 订单包含的商品信息                     |
| 订单金额   | 订单总金额                             |
| 订单状态   | 订单当前状态                           |
| 支付状态   | 支付状态                               |
| 创建时间   | 订单创建时间                           |
| 操作       | 查看详情、确认订单、取消订单、申请退款 |

### 5.2 订单详情

#### 5.2.1 查看订单详情

1. 在订单列表中找到需要查看的订单
2. 点击"查看详情"按钮
3. 进入"订单详情"页面

#### 5.2.2 订单详情内容

**支付信息**：

- 支付方式
- 支付时间

**配送信息**：

- 配送时间
- 配送人员
- 配送电话

**备注信息**：

- 订单备注
- 商家备注

### 5.3 确认订单

#### 5.3.1 确认订单步骤

1. 在订单列表中找到待确认的订单
2. 点击"确认订单"按钮
3. 系统弹出确认对话框
4. 填写商家备注（可选）
5. 点击"确定"按钮
6. 系统提示确认成功
7. 订单状态变为"待支付"

#### 5.3.2 确认订单注意事项

- 确认前请仔细核对订单信息
- 确认后订单将进入支付流程
- 如有异常情况，请联系客户
- 建议及时确认订单（建议10分钟内）

### 5.4 取消订单

#### 5.4.1 取消订单步骤

1. 在订单列表中找到需要取消的订单
2. 点击"取消订单"按钮
3. 系统弹出取消对话框
4. 选择取消原因：
   - 用户主动取消
   - 商品缺货
   - 配送范围外
   - 其他原因
5. 填写取消说明（可选）
6. 点击"确定"按钮
7. 系统提示取消成功
8. 订单状态变为"已取消"

#### 5.4.2 取消订单限制

- 已支付的订单需要先退款才能取消
- 配送中的订单不能取消
- 已完成的订单不能取消

### 5.5 申请退款

#### 5.5.1 申请退款步骤

1. 在订单列表中找到需要退款的订单
2. 点击"申请退款"按钮
3. 系统弹出退款对话框
4. 选择退款类型：
   - 全额退款
   - 部分退款
5. 填写退款金额
6. 选择退款原因：
   - 商品问题
   - 配送问题
   - 服务问题
   - 其他原因
7. 填写退款说明（可选）
8. 点击"提交"按钮
9. 系统提示提交成功
10. 等待审核（1-3个工作日）

#### 5.5.2 退款审核流程

1. 用户提交退款申请
