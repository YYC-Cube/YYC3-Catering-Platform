## 📖 模块概述

### 模块简介

工作台是YYC³餐饮行业智能化平台管理后台的核心模块，为用户提供一站式的数据监控和操作入口。工作台整合了实时数据看板、快捷操作、待办事项、通知中心等功能，帮助用户快速了解系统运营状态并高效完成日常工作。

### 核心价值

- 🚀 **高效决策**：实时数据展示，快速掌握运营状况
- 🎯 **快速操作**：常用功能快捷入口，提升操作效率
- 📋 **任务管理**：待办事项管理，确保重要任务不遗漏
- 🔔 **即时通知**：系统通知中心，及时获取重要信息

### 技术栈

- **前端框架**：Vue 3 + TypeScript
- **UI组件库**：Element Plus
- **状态管理**：Pinia
- **图表库**：ECharts
- **实时通信**：WebSocket
- **构建工具**：Vite

## 🔌 核心功能

### 1. 实时数据看板

#### 功能描述

展示系统核心运营数据，包括今日订单、销售额、客户数等关键指标，支持数据实时更新和趋势分析。

#### 技术实现

```typescript
// 核心指标数据获取
async function loadCoreMetrics() {
  const response = await dashboardApi.getCoreMetrics({
    startDate: dateRange.value[0],
    endDate: dateRange.value[1],
  });

  if (response.success) {
    dashboardStore.setCoreMetrics(response.data);
  }
}

// WebSocket实时更新
function handleWebSocketMessage(data: any) {
  switch (data.type) {
    case "new_order":
      dashboardStore.addOrder(data.payload);
      break;
    case "order_updated":
      dashboardStore.updateOrderStatus(data.payload.id, data.payload.status);
      break;
  }
}
```

#### 性能优化

- 使用WebSocket实现数据实时推送，减少轮询开销
- 数据缓存机制，避免重复请求
- 虚拟滚动优化大数据量列表渲染
- 图表懒加载，按需渲染

// 智能推荐算法
function getRecommendedActions(): QuickAction[] {
  const usageStats = quickActionsStore.usageStats;

  return defaultActions
    .sort((a, b) => {
      const countA = usageStats.get(a.id) || 0;
      const countB = usageStats.get(b.id) || 0;
      return countB - countA;
    })
    .slice(0, 8);
}

### 3. 待办事项管理

#### 功能描述

管理用户的待办事项，支持添加、编辑、删除、完成、筛选、排序等功能，提供逾期提醒和分类管理。

#### 技术实现

```typescript
// 待办事项数据结构
interface Todo {
  id: string;
  title: string;
  description?: string;
  status: TodoStatus; // pending | in-progress | completed
  priority: TodoPriority; // high | medium | low
  category?: string;
  dueDate?: string;
  reminder: boolean;
  createdAt: string;
  updatedAt: string;
}

// 待办事项状态切换
async function toggleTodoStatus(todo: Todo) {
  const newStatus = todo.status === "completed" ? "pending" : "completed";

  await todoApi.updateTodo(todo.id, { status: newStatus });

  todo.status = newStatus;

  // 如果有提醒，发送通知
  if (newStatus === "completed" && todo.reminder) {
    notificationApi.send({
      title: "待办完成",
      message: `您已完成待办：${todo.title}`,
      type: "success",
    });
  }
}

// 逾期检查
function checkOverdueTodos() {
  const now = new Date();

  todos.value.forEach(todo => {
    if (todo.dueDate && todo.status !== "completed") {
      const dueDate = new Date(todo.dueDate);

      if (dueDate < now) {
        notificationApi.send({
          title: "待办逾期",
          message: `您的待办"${todo.title}"已逾期`,
          type: "warning",
        });
      }
    }
  });
}
```

#### 功能特性

- 拖拽排序
- 优先级和分类管理
- 逾期提醒
- 批量操作
- 导出功能
- 本地存储同步

### 4. 通知中心

#### 功能描述

集中展示系统通知、待办事项、系统消息等，支持实时推送、分类查看、已读标记等功能。

#### 技术实现

```typescript
// 通知数据结构
interface Notification {
  id: string;
  title: string;
  message: string;
  type: NotificationType;
  status: NotificationStatus; // unread | read | archived
  priority: "low" | "medium" | "high" | "urgent";
  data?: any;
  actions?: Array<{
    id: string;
    label: string;
    style?: string;
    url?: string;
    action?: string;
  }>;
  createdAt: string;
}

// WebSocket通知推送
function connectWebSocket() {
  const ws = new WebSocket(`${WS_URL}/notifications`);

  ws.onmessage = event => {
    const notification = JSON.parse(event.data);

    notificationStore.addNotification(notification);

    // 显示浏览器通知
    if (Notification.permission === "granted") {
      new Notification(notification.title, {
        body: notification.message,
        icon: "/favicon.ico",
        tag: notification.id,
      });
    }

    // 播放提示音
    playNotificationSound();
  };
}
```

#### 功能特性

- 实时推送
- 分类查看
- 已读/未读状态
- 批量操作
- 浏览器通知
- 提示音效
- 通知设置
