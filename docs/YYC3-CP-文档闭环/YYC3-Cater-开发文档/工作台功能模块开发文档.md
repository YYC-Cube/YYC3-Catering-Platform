## 📖 模块概述

### 模块简介

工作台是YYC³餐饮行业智能化平台管理后台的核心模块，为用户提供一站式的数据监控和操作入口。工作台整合了实时数据看板、快捷操作、待办事项、通知中心等功能。

### 核心功能

- 📊 **实时数据看板**：展示核心运营数据和趋势分析
- ⚡ **快捷操作**：提供常用功能的快速入口
- 📝 **待办事项**：管理日常工作任务
- 🔔 **通知中心**：集中展示系统通知

### 技术栈

- **前端框架**：Vue 3 + TypeScript
- **UI组件库**：Element Plus
- **状态管理**：Pinia
- **图表库**：ECharts
- **实时通信**：WebSocket
- **构建工具**：Vite
- **测试框架**：Vitest

### 状态管理模式

#### Store结构

```typescript
// stores/dashboard.ts
import { defineStore } from "pinia";

export const useDashboardStore = defineStore("dashboard", {
  state: () => ({
    coreMetrics: {} as CoreMetrics,
    revenueData: [] as RevenueData[],
    orderDistribution: [] as OrderDistribution[],
    topDishes: [] as TopDish[],
    customerFlow: [] as CustomerFlow[],
    recentOrders: [] as Order[],
    loading: false,
    error: null as string | null,
  }),

  getters: {
    totalRevenue: state => state.coreMetrics.totalRevenue || 0,
    totalOrders: state => state.coreMetrics.totalOrders || 0,
    pendingOrdersCount: state => {
      return state.recentOrders.filter(o => o.status === "pending").length;
    },
  },

  actions: {
    async loadDashboardData(params: any) {
      this.loading = true;
      try {
        const response = await dashboardApi.getDashboardData(params);
        if (response.success && response.data) {
          this.coreMetrics = response.data.coreMetrics || {};
          this.revenueData = response.data.revenueData || [];
          this.orderDistribution = response.data.orderDistribution || [];
          this.topDishes = response.data.topDishes || [];
          this.customerFlow = response.data.customerFlow || [];
          this.recentOrders = response.data.recentOrders || [];
        }
        return response;
      } catch (error: any) {
        this.error = error.message || "Failed to load dashboard data";
        throw error;
      } finally {
        this.loading = false;
      }
    },

    addOrder(order: Order) {
      this.recentOrders.unshift(order);
      if (this.recentOrders.length > 10) {
        this.recentOrders.pop();
      }
    },

    updateOrderStatus(orderId: string, status: string) {
      const index = this.recentOrders.findIndex(order => order.id === orderId);
      if (index !== -1) {
        this.recentOrders[index].status = status;
      }
    },
  },
});
```

## 🔍 故障排查

### 常见问题

#### 1. 数据加载失败

**问题**：工作台数据无法加载

**排查步骤**：

1. 检查网络连接
2. 检查API地址配置
3. 检查后端服务状态
4. 查看浏览器控制台错误

**解决方案**：

```typescript
// 添加错误处理
try {
  await dashboardStore.loadDashboardData();
} catch (error) {
  console.error("Load dashboard data failed:", error);
  ElMessage.error("数据加载失败，请稍后重试");
}
```

#### 2. WebSocket连接失败

**问题**：WebSocket无法连接

**排查步骤**：

1. 检查WebSocket地址配置
2. 检查防火墙设置
3. 检查后端WebSocket服务
4. 查看网络请求日志

**解决方案**：

```typescript
// 添加重连机制
function connectWebSocket() {
  const ws = new WebSocket(WS_URL);

  ws.onopen = () => {
    console.log("WebSocket connected");
  };

  ws.onerror = error => {
    console.error("WebSocket error:", error);
    setTimeout(connectWebSocket, 5000);
  };
}
```

#### 3. 组件渲染异常

**问题**：组件无法正常渲染

**排查步骤**：

1. 检查组件导入
2. 检查Props传递
3. 检查数据格式
4. 查看Vue DevTools

**解决方案**：

```typescript
// 添加类型检查

// 添加默认值
```
